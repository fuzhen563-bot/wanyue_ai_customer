嵌入文档

将万岳AI智能客服嵌入到您的网站

快速开始 嵌入方式 配置参数 常见问题

快速开始

只需3步，即可将AI客服嵌入您的网站：

1

获取嵌入代码

登录后进入管理后台 → 嵌入配置 → 创建配置 → 获取嵌入代码

2

复制代码

复制系统生成的嵌入代码，添加到您网站HTML的 <body> 标签结束前

3

完成

保存页面，刷新浏览器，即可看到AI客服图标

嵌入方式

方式一：标准嵌入（推荐）

最简单的方式，只需一行代码即可完成嵌入：

<!-- 万岳AI客服 - 开始 -->
<script>
  window.wanyueConfig = {
    authCode: "您的授权码",
    apiUrl: window.location.origin
  };
</script>
<script src="/static/embed.js" async></script>
<!-- 万岳AI客服 - 结束 -->

方式二：自定义配置

您可以通过配置参数自定义客服的外观和行为：

<script>
  window.wanyueConfig = {
    authCode: "您的授权码",
    apiUrl: window.location.origin,
    // 自定义配置
    position: "right",      // 位置: left/right
    themeColor: "#00c7be", // 主题颜色
    welcomeMessage: "您好，请问有什么可以帮助？", // 欢迎语
    height: "600px",       // 聊天窗口高度
    offsetX: "20px",      // 水平偏移
    offsetY: "20px"       // 垂直偏移
  };
</script>
<script src="/static/embed.js" async></script>

方式三：React/Vue 中使用

在现代前端框架中使用：

// React 组件中使用
useEffect(() => {
  const script = document.createElement('script');
  script.src = window.location.origin + '/static/embed.js';
  script.async = true;
  document.body.appendChild(script);

  window.wanyueConfig = {
    authCode: "您的授权码",
    apiUrl: window.location.origin
  };

  return () => {
    document.body.removeChild(script);
  };
}, []);
// Vue 3 组件中使用
onMounted(() => {
  const script = document.createElement('script');
  script.src = window.location.origin + '/static/embed.js';
  script.async = true;
  document.body.appendChild(script);

  window.wanyueConfig = {
    authCode: "您的授权码",
    apiUrl: window.location.origin
  };
});

配置参数

所有支持的配置选项：

参数 类型 默认值 说明
authCode string - 必填，您的授权码
apiUrl string window.location.origin API服务器地址
position string "right" 悬浮图标位置: left/right
themeColor string "#00c7be" 主题颜色
welcomeMessage string "您好！我是..." 首次欢迎语
height string "600px" 聊天窗口高度
offsetX string "20px" 水平偏移量
offsetY string "20px" 垂直偏移量

常见问题

Q1: 嵌入代码放在哪里？

建议放在 HTML 页面的 <body> 标签结束前，这样可以确保页面加载完成后再加载客服组件。

Q2: 需要配置域名吗？

是的，为了安全起见，您需要在嵌入配置中设置允许的域名。只有在白名单域名下才能正常加载。

Q3: 客服图标不显示怎么办？

请检查以下几点：

1. 确认 authCode 正确

2. 检查域名是否在白名单

3. 查看浏览器控制台是否有错误

Q4: 如何自定义客服样式？

可以通过 themeColor 参数设置主题颜色，或通过 CSS 覆盖默认样式。

Q5: 支持 HTTPS 吗？

支持！建议使用 HTTPS 协议，确保数据传输安全。
